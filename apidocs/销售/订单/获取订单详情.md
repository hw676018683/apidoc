# 获取订单详情
## GET /sales/order/detail/(\d+)

## Path 参数说明
- ID: 订单ID

## Query 参数说明
- name: 用户名

## 返回体说明
```json5
{
  "code": "ok",
  "msg": "",
  "data": {
    "name": "",                                   // 名称
    "age": "",                                    // 年龄
    "res": ""                                     // 返回信息
  }
}
```
